# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-07-09

### Added
- Initial release of InterviewAce.
- User authentication via Clerk.
- AI-powered mock interviews using Google Gemini.
- Resume parsing feature for tailored questions.
- Performance visualization using Recharts.
- Basic SEO implementation with Open Graph and JSON-LD.
- Continuous Integration workflow via GitHub Actions.
